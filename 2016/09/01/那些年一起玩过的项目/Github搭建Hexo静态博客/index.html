静态博客的优势

hexo相对于wp，最大的区别就是它是一个静态博客。静态博客有什么特点？

  • 不用配置服务器
  • 不用数据库
  • 访问速度相当快
  • 没有安全性可言
  • 使用户可更注重博客内容
  • 支持markdown

注意：
在github上建立的个人静态页面repo的名字必须与你的username相同，否则github不会自动生成静态页面。比如thuyangyu/aldrich_yang.github.io不会自动生成页面，即此github.io的地址不能被访问。
