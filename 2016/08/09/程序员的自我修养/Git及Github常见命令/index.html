建立自己的Github的repo并把自己的代码push上去，生成公开链接

###后续问题
我现在git clone下来了，如何push到自己的github主页上并生成一个作业链接？

###二师兄解决方案：
你在你的github上新建一个repo，然后clone到本地，然后将你本地写好的代码文件夹复制到本地的指针当中，然后再push上去。

###自己使用：
与当时计原张艺庆协作过程回顾：为了避免进行merge操作，每次都从服务器上pull下来最新的project修改完了commit再push上去，这样只要修改工程的几个人不同时工作，就不会出现需要merge的问题。
