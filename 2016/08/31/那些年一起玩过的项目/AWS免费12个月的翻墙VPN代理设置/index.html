进行弹性IP绑定，获得实例即时重启也还保留原来的IP地址。

在mac或者ubuntu的ssh终端执行命令：
sudo ssh –i vpn.pem ubuntu@54.65.230.174
其中vpn.pem是创建aws实例时候下载的密钥文件，成功建立ssh连接。

(当多次连接之后有可能出现的密钥pem错误的一个问题就是
Offending ECDSA key in /home/aldrich/.ssh/known_hosts:1
remove with: ssh-keygen -f “/home/aldrich/.ssh/known_hosts” -R 54.65.230.174
ECDSA host key for 54.65.230.174 has changed and you have requested strict checking.)
使用提示的命令ssh-keygen -f "/home/aldrich/.ssh/known_hosts" -R 54.65.230.174解决即可。

关于AWS服务器允许所有入站出站连接的设置方法：

  1. 创建安全组（Security Group），添加组名称和描述，编辑入站和出站规则（两者都要类似编辑），类型为所有流量，来源为任何位置。
  2. 选择正在运行的实例右键-联网，更改安全组，选择刚才设置的安全组。
  3. 实例-右键-联网-禁用源/目标检查，设置为禁用，ok服务器设置配置完成。
