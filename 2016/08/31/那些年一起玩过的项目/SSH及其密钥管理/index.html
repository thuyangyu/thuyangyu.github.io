#SSH及其密钥管理

##SSH的原理
SSH是Secure Shell的缩写，由IETF的网络工作小组（Network Working Group）所制定,是一种专为远程登录会话和其他网络服务提供安全性的协议。利用 SSH 协议可以有效防止远程管理过程中的信息泄露问题。

###ssh传输的过程如下:

  1. 远程主机收到用户的登录请求，把自己的公钥发给用户。
  2. 用户使用这个公钥，将登录密码加密后，发送回来。
  3. 远程主机用自己的私钥，解密登录密码，如果密码正确，允许用户登录。

##在自己电脑上安装ssh-server
ubuntu,debian:
sudo apt-get install openssh-server
centos:
yum install openssh-server
一般网上申请或购买的云服务器已经自带ssh-server了，在远程登陆自己实验室的服务器，配置时需手动安装一下服务器端程序。

##ssh客户端

  1. Mac和linux命令行自带ssh客户端，当然UNIX内核的系统上有iterm2这个更加功能强大的命令行软件；
  2. Windows推荐使用putty软件来进行ssh远程登陆（putty client支持导入密钥等功能），或者可以使用openssh-for-windows来远程登陆。
