Learn More

Miirabella is a makeup assistant app that could give you instructions as you are putting on makeups. Using computer vision & machine learning technology, Miirabella is able to guide you through the makeup process step by step. You will open Miirabella app, preview all the recommended looks and select the desired one, then start the interactive makeup tutorial.

 

Live Preview

Miirabella, a smart cosmetic assistant, that aims to help beauty lovers to rock any digital look into reality. It provides real-time interactive tutorials that are tailored to meet different individual needs. Prior to the chosen tutorial, all the recommended or self-preferred digital makeovers can be previewed and tweaked.

The first to support customizable tutorials in cosmetics with live preview feature.

Product try on

Miirabella allows users to try on products without make a purchase. Thanks to the depth-camera function and 3D facial recognition technology, Miirabella gives users true color reflection on the whole face while applying makeup.


The first to add white-balance lighting control in cosmetics using true 3D facial strucutre.

Interactive Tutorial

Miirabella understands the real struggle when getting the makeup done. All the interactive tutorials Miirabella offers are specially designed to assist beauty lovers to apply cosmetics every step of the way. The real-time feedback makes the process as easy as snap a selfie.

The first to offer fully interactive tutorials with real-time feedback and corrections.

From digital to reality

 

Since the selfie era began, many mobile applications have been developed to help their users get real-time digital makeovers using different makeup products and shade combinations. These users can then take selfies of the new looks to share with friends. Unfortunately, the effects only stay digitally on the photos, not in reality.

With our product, Miirabella, we can help you take these pretty looks from digital into reality by giving users on-face instructions and real-time feedback to assist them on applying makeup.
