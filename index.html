#SSH及其密钥管理

##SSH的原理
SSH是Secure Shell的缩写，由IETF的网络工作小组（Network Working Group）所制定,是一种专为远程登录会话和其他网络服务提供安全性的协议。利用 SSH 协议可以有效防止远程管理过程中的信息泄露问题。

###ssh传输的过程如下:

  1. 远程主机收到用户的登录请求，把自己的公钥发给用户。
  2. 用户使用这个公钥，将登录密码加密后，发送回来。
  3. 远程主机用自己的私钥，解密登录密码，如果密码正确，允许用户登录。

##在自己电脑上安装ssh-server
ubuntu,debian:
sudo apt-get install openssh-server
centos:
yum install openssh-server
一般网上申请或购买的云服务器已经自带ssh-server了，在远程登陆自己实验室的服务器，配置时需手动安装一下服务器端程序。

##ssh客户端

  1. Mac和linux命令行自带ssh客户端，当然UNIX内核的系统上有iterm2这个更加功能强大的命令行软件；
  2. Windows推荐使用putty软件来进行ssh远程登陆（putty client支持导入密钥等功能），或者可以使用openssh-for-windows来远程登陆。

Moto X Style 刷机教程

Posted on 2016-08-19   |   In 那些年一起玩过的项目

#Moto X Style 刷机教程

##刷机的动力
刷机的根本目的：
据说亚太版比国行更加稳定流畅，区别有：

  1. 国行rom微信切换界面卡的情况，在亚太rom不存在。
  2. 国行rom卡顿掉帧的情况，在亚太rom不存在。整体运行顺畅度和跟指的速度，亚太rom完胜国行rom，极少掉帧。
  3. 国行rom插入充电器后手机卡顿的情况，在亚太rom不存在。
  4. 亚太版对google服务有良好支持。

##第一步、解锁bootloader

  1. Windows电脑安装手机驱动，
  2. Windows电脑下载Fastboot Tools 工具（windows版）
  3. Moto手机在设置中连续点击7次“关于手机-版本号”，打开开发者选项，在开发者选项中打开“允许usb调试”和允许“OEM解锁”，如图
  4. Moto手机关机按住电源键和音量减3s进入fastboot模式，连接电脑
  5. 运行fastboot devices监测是否成功连接

参考链接 Moto X Style 解锁教程简洁版 Moto X Style 解锁教程完整版

  1. 如果上一步连接成功，接着运行fastboot oem get_unlock_data,并将返回的代码拼成一个字符串待用。
  2. 登陆摩托罗拉官方解锁页面,按照引导用某个邮箱注册一个摩托罗拉账户。
  3. 注册完成自动跳转到刚才的解锁引导页面，按照网站提供的操作步骤输入6步中的返回代码。点击“Can my device can be unlocked”
  4. 下面点击Agree协议，并获取Request unlock key
  5. 查收邮件获取unlock key
  6. 在fastboot命令行中输入fastboot oem unlock UNIQUE_KEY，其中的UNIQUE_KEY就是邮件中获取的key


解锁成功后手机上显示的Device is Locked一行变为黄色，内容变为Device is UNLOCKED. Status Code: 3

##第二步及以后
机锋论坛这个帖子
汇总了后面刷机的所有说明，这里简单罗列一下之后的步骤。

说明：刷亚太系统的过程中论坛上的人各种不建议刷新的bootloader和更新版的基带，据说是这两样东西更新了就回不来了，会被摩托公司限制。因此从国行版本的5.1.1升级到亚太版的6.0系统以上就大至需要采取以下方法。

  1. 下载亚太6.0以上系统双卡的底包
  2. 下载国行5.1.1系统的底包，提取其中的fsg.mbn和NON-HLOS.bin,一个是基带文件一个是射频文件，替换掉亚太6.0以上系统中对应的文件进行刷入。( 如果进去以后wifi不能启动，重新刷一下6.0.1里面的 NON-HLOS.bin，命令式：fastboot flash modem NON-HLOS.bin就可以完美解决。)
  3. 刷入logo，去掉如下的开机解锁警告画面。（可以修改一键刷机的脚本文件把下面的刷logo的脚本一起在上一步就刷进去）fastboot flash logo img/logo.bin img文件夹中的logo.bin文件是6.0自带的原生logo，超级漂亮，StyleLogo.bin是论坛上人自制的简化版的logo
  4. 刷入recovery，注意：刷完后必须马上进入Recovey，在Rec里面点击reboot system，才能生效，如果刷完不进入REC，直接就重启手机，则REC会自动失效变回原版！
  5. 刷入root，root就是刷入supersu包（现在开始用线刷，记得以前HTC华为什么的都是用卡刷的），上面一步的刷入recovery就是为了这一步用recovery卡刷root包用的。手机开机连接电脑，将supersu压缩包放入手机根目录，重启到recovery开始install。安装好的supersu就是系统里面的一个应用程序了。
    6.卡刷安装cdma电信补丁包，从亚太的联通移动双网通变成支持电信的三网通（如果人家国行的机子默认支持三网通，那么刷机之后也是只要解决了软件问题，一定也可以支持三网通的）。这个包里还有google messager默认短信应用，moto控制中心（国行版本自带的软件）。一键解决非常方便。

##翻墙需求背景
之前使用的是Amazon的AWS进行翻墙服务，注册一个Amazon的AWS账号，赠送一个免费一年的虚拟主机，在上面安装了openvpn进行代理。存在下面三个重要的问题：

  • AWS只能使用教育网连接，在学校没问题，但在家里的电信网（移动网、联通网）均不能直接访问到AWS上自己配置的openvpn服务器。
  • openvpn这个软件最近存在一些不稳定的情况，及不论是手机还是电脑，经常只能连接3分钟然后就自动短线，需要手工重连。有时候手动重连依然连接不上，这种情况在看youtube视频时尤其明显。
  • Amazon AWS免费虚拟主机一年期到期了，在申请新的虚拟主机明显太贵，相比于其它VPS不具有价格优势。

##新VPS的选择问题
朋友推荐，当前最流行的3个VPS，Linode，DigitalOcean，Vultr。网上有很多关于这三种VPS服务质量的评测，套餐价格等的讨论。综合来看Linode出现最早，是老牌VPS，功能什么的自不必说。后两者DO和Vultr的优势在于最低一档的套餐价格比Linode更便宜，后两者最低价的套餐都是$5元每月，而Linode的最低套餐是$10元每月。相对而言，DO使用服务器对电信，移动等网络的支持不够好，最终选择了Vultr的VPS服务器，可以保证在学校，在家，包括使用手机上网时都能随时随地地翻墙。
