在mac或者ubuntu的ssh终端执行命令：
sudo ssh –i vpn.pem ubuntu@54.65.230.174
其中vpn.pem是创建aws实例时候下载的密钥文件，成功建立ssh连接。

remove with: ssh-keygen -f “/home/aldrich/.ssh/known_hosts” -R 54.65.230.174

使用提示的命令ssh-keygen -f "/home/aldrich/.ssh/known_hosts" -R 54.65.230.174解决即可。

  3. 实例-右键-联网-禁用源/目标检查，设置为禁用，ok服务器设置配置完成。

Moto X Style 刷机教程

Veröffentlicht am 2016-08-19   |   in 那些年一起玩过的项目

#Moto X Style 刷机教程

##刷机的动力
刷机的根本目的：
据说亚太版比国行更加稳定流畅，区别有：

  1. 国行rom微信切换界面卡的情况，在亚太rom不存在。
  2. 国行rom卡顿掉帧的情况，在亚太rom不存在。整体运行顺畅度和跟指的速度，亚太rom完胜国行rom，极少掉帧。
  3. 国行rom插入充电器后手机卡顿的情况，在亚太rom不存在。
  4. 亚太版对google服务有良好支持。

##第一步、解锁bootloader

  1. Windows电脑安装手机驱动，
  2. Windows电脑下载Fastboot Tools 工具（windows版）
  3. Moto手机在设置中连续点击7次“关于手机-版本号”，打开开发者选项，在开发者选项中打开“允许usb调试”和允许“OEM解锁”，如图
  4. Moto手机关机按住电源键和音量减3s进入fastboot模式，连接电脑
  5. 运行fastboot devices监测是否成功连接

参考链接 Moto X Style 解锁教程简洁版 Moto X Style 解锁教程完整版

  1. 如果上一步连接成功，接着运行fastboot oem get_unlock_data,并将返回的代码拼成一个字符串待用。
  2. 登陆摩托罗拉官方解锁页面,按照引导用某个邮箱注册一个摩托罗拉账户。
  3. 注册完成自动跳转到刚才的解锁引导页面，按照网站提供的操作步骤输入6步中的返回代码。点击“Can my device can be unlocked”
  4. 下面点击Agree协议，并获取Request unlock key
  5. 查收邮件获取unlock key
  6. 在fastboot命令行中输入fastboot oem unlock UNIQUE_KEY，其中的UNIQUE_KEY就是邮件中获取的key


解锁成功后手机上显示的Device is Locked一行变为黄色，内容变为Device is UNLOCKED. Status Code: 3

##第二步及以后
机锋论坛这个帖子
汇总了后面刷机的所有说明，这里简单罗列一下之后的步骤。

说明：刷亚太系统的过程中论坛上的人各种不建议刷新的bootloader和更新版的基带，据说是这两样东西更新了就回不来了，会被摩托公司限制。因此从国行版本的5.1.1升级到亚太版的6.0系统以上就大至需要采取以下方法。

  1. 下载亚太6.0以上系统双卡的底包
  2. 下载国行5.1.1系统的底包，提取其中的fsg.mbn和NON-HLOS.bin,一个是基带文件一个是射频文件，替换掉亚太6.0以上系统中对应的文件进行刷入。( 如果进去以后wifi不能启动，重新刷一下6.0.1里面的 NON-HLOS.bin，命令式：fastboot flash modem NON-HLOS.bin就可以完美解决。)
  3. 刷入logo，去掉如下的开机解锁警告画面。（可以修改一键刷机的脚本文件把下面的刷logo的脚本一起在上一步就刷进去）fastboot flash logo img/logo.bin img文件夹中的logo.bin文件是6.0自带的原生logo，超级漂亮，StyleLogo.bin是论坛上人自制的简化版的logo
  4. 刷入recovery，注意：刷完后必须马上进入Recovey，在Rec里面点击reboot system，才能生效，如果刷完不进入REC，直接就重启手机，则REC会自动失效变回原版！
  5. 刷入root，root就是刷入supersu包（现在开始用线刷，记得以前HTC华为什么的都是用卡刷的），上面一步的刷入recovery就是为了这一步用recovery卡刷root包用的。手机开机连接电脑，将supersu压缩包放入手机根目录，重启到recovery开始install。安装好的supersu就是系统里面的一个应用程序了。
    6.卡刷安装cdma电信补丁包，从亚太的联通移动双网通变成支持电信的三网通（如果人家国行的机子默认支持三网通，那么刷机之后也是只要解决了软件问题，一定也可以支持三网通的）。这个包里还有google messager默认短信应用，moto控制中心（国行版本自带的软件）。一键解决非常方便。

##扩展玩机
安装xposed框架。
刷入Moto X Style对应的arm zip包，然后重启手机， 在手机上运行XposedInstaller主程序，进行框架安装和初始化调整。

Farbox搭建最简单的个人网站

Veröffentlicht am 2016-08-09   |   in 那些年一起玩过的项目

Farbox，正如官方所说，果然是一见倾心的写作工具

Farbox就是强调最简单的博客，基于Markdown编写md文件的笔记文档，然后自动同步到服务器端，然后自动生成你的博客页面。强调的是用类似于网盘的形式管理你的博客

#开始搭建步骤
使用Windows，Mac或者Linux系统，下载Farbox Editor客户端，注册一个简单的Basic账户（前29天免费），在本地建立一个文件夹，比如我的在Mac上建立的文件夹的地址是/Users/Aldrich/Documents/FarBox/yangyu-aldrich.farbox.com,在这个文件夹地址之下建立你博客文章简单的分类，比如：


然后在对应的分类文件夹下写文章（关于文章的书写，见下一节），这样在同步之后你网站上分类这一列表里面就回显示你按照上述文件夹分的类，便于本地与云端相互管理。

##使用Farbox写文章

设置Farbox，建议将新建的文章后缀名设置为默认的.md文件，这样就可以直接用markdown语法来写文章了。

###在Farbox中创建文档
使用Farbox Editor，在你的某个分类文件夹下新建一个post，生成以当天日期为标题的一个post，我们可以修改标题为我们的post需要的标题。使用Farbox Editor新建的post文本文件中会自动显示文件头信息：

以上信息需要在本地找到该文件的位置，使用其它的文本编辑器打开（比如Sublime Text）才能看的到，使用Farbox Editor本身是看不到的。在修改文章标题之后对应的文件信息的title部分会自动修改。

###整理自己的其它笔记
整理自己使用其它markdown编辑器书写的笔记最好在Farbox Editor中新建一个post然后复制过来，对应的图片重新插入；如果你原来写的笔记图片较多的话也可以直接复制到本机上Farbox文章分类对应的文件夹下，使用其它文本编辑器(如Sublime Text)打开并在开头插入上小节中的文件头信息；创建对应的_image文件夹和其下的以文章标题命名的子文件夹，并在置入你的图片，在该目录下的图片，可以在你原来的md文件中将其链接修改如下形式即可。然后同步你的笔记即可。

###在Farbox中插入图片
Farbox Editor是我见过的对插入图片支持最好的Markdown文件编辑器，你可以直接在自己电脑屏幕上截图到剪贴板，然后直接在Farbox Editor编辑某篇post过程中粘贴该图片，图片自动存储到如下所示的位置，并在post中自动生成基于markdown语法的链接，而且你不必担心，这些图片全部可以同步到云端。

##日常维护
需要经常在www.farbox.com的个人信息一栏当中查看当月访问数和已用流量，考虑是否需要升级套餐。Basic的月流量每月只有100MB，付费基本版每月2GB流量，年费65元。

##微信文章收藏到网站

  • 在Farbox公众号里面文本输入框中点击加号，我的收藏，直接把收藏的文章发过来，并等待3分钟，知道本地文件夹里面出现了该公众号的文章（在根目录中）。
  • 在本地文件夹中操作，将得到的txt文章移动到“我的微信收藏”文件夹中。
  • 在Farbox编辑器中打开该txt文件，并将文章末尾的Source From移动到最前面。
  • 微信文件传输助手保存该文章中的所有图片到本地，再重新在Farbox中插入图片。
  • 同步所有文件，完成收藏别人公众号的文章。
